# vue-delay-clock
基于 vue.js 和 moment.js 实现，简单的、可延迟、动态时间展示组件。
[npm](https://www.npmjs.com/package/vue-delay-clock)

## usage

### install
```sh
yarn add vue moment
yarn add vue-delay-clock
```

### use
```js
import VueDelayClock from 'vue-delay-clock'
```
